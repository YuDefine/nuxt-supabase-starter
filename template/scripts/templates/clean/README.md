# {{PROJECT_NAME}}

Built with [Nuxt Supabase Starter](https://github.com/YuDefine/nuxt-supabase-starter).

## Stack

Nuxt 4 · Vue 3 · TypeScript · Tailwind CSS · Nuxt UI · Pinia · Supabase · Cloudflare Workers

## Getting Started

```bash
pnpm install
pnpm skills:install
cp .env.example .env  # 填入環境變數
supabase start
pnpm dev
```

## Commands

| Command             | Description                      |
| ------------------- | -------------------------------- |
| `pnpm dev`          | 啟動開發伺服器                   |
| `pnpm check`        | Lint + Format + Test + Typecheck |
| `pnpm build`        | 建置                             |
| `supabase db reset` | 重置資料庫                       |
